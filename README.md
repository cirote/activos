## Base para el sistema de Activos

Proyecto para para el seguimiento de activos financieros.

### Instalacion

Via [Composer](http://getcomposer.org/). 

``` bash
$ composer require cirote/activos
```

Realizar las migraciones

``` bash
$ php artisan migrate
```

### Utilizacion

Via web

``` bash
https://yoursite.com/
```
